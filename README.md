## Assignment Details

### Assignment 1: Library Management System
- **Data Structure Used:** Arrays / Linked List  
- **Operations Included:**  
  - Add book  
  - Delete book  
  - Search book  
  - Display all books  
- **Concepts Covered:**  
  - Data storage  
  - Menu-driven programs  
  - Basic data manipulation  

---

### Assignment 2: Code Club Subscription
- **Data Structure Used:** Singly Linked List  
- **Operations Included:**  
  - Insert member  
  - Delete member  
  - Search member  
  - Display subscriber list  
- **Concepts Covered:**  
  - Dynamic memory allocation  
  - Node creation  
  - Traversal of linked list  

---

### Assignment 3: Music System
- **Data Structure Used:** Circular Doubly Linked List (CDLL)  
- **Operations Included:**  
  - Add song  
  - Delete song  
  - Play next song  
  - Play previous song  
  - Display playlist  
- **Concepts Covered:**  
  - Circular linking  
  - Bidirectional traversal  
  - Efficient navigation  

---

### Assignment 4: Subset Sum Problem

- **Concepts Covered:**  
  - Stack operations  
  - Backtracking  
  - Recursive problem solving
